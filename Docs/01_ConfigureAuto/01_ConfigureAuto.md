# Exercise 01: Configure autonomous agent settings in Customer Service


## Introduction
With high volumes of case-status checks, warranty questions, and service issues, Contoso needs consistent, automated case handling that still fits their global support model. In this exercise, you'll configure the Case Management Agent's autonomous capabilities so it can help drive faster case creation, updates, follow-up, closure, and resolution-reducing manual effort while improving first-contact and end-to-end resolution outcomes.
