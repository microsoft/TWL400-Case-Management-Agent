## Task 02: Configure global settings


### Introduction
Contoso needs the Case Management Agent to operate consistently across channels and teams, so it must have a trusted identity to act on behalf of the organization. Setting the application user establishes the baseline for secure, traceable automation.

### Description
You'll open Copilot Service admin center, go to the Case Management Agent settings, and assign the Case Agent application user under Global Settings so the agent can perform autonomous actions.

### Success criteria
- The Case Management Agent Global Settings are saved with the Case Agent application user assigned.

### Key steps


1. Open the **Copilot Service admin center** app.

	![6pydptu6.jpg](../../media/6pydptu6.jpg)

1. In the left pane, in the **Customer support** section, select **Case Settings**.

	![nkkfbn7x.jpg](../../media/nkkfbn7x.jpg)

1. Locate **Case Management Agent** and then select **Manage**.

    ![exzytddc.jpg](../../media/exzytddc.jpg)

1. In the **Global Settings** section, select **Manage**.

1. Below **Set an application user for the case management agent**, in the **Application user** field, select the **Case Agent** application user you created earlier.

1. On the command bar, select **Save**.
